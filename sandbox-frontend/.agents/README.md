# Sandbox Frontend

Modern React application for the SeedGPT sandbox demo interface.

## Purpose
Provides an intuitive UI where users can describe project ideas and watch AI generate complete project structures in real-time.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **WebSocket**: Native WebSocket API
- **State**: React Hooks

## Key Features
- Real-time progress visualization
- WebSocket-based live updates
- Responsive design
- Modern UI/UX
- Error handling and retry logic
