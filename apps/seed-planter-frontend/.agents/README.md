# Seed Planter Frontend

Modern React application for planting and growing autonomous AI-driven projects.

## Purpose
Allows users to "plant" project seeds that grow into permanent, autonomous projects. Users provide a project name and description, then watch in real-time as AI creates a GitHub organization, forks SeedGPT template, customizes the project, sets up GCP, and deploys.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **WebSocket**: Native WebSocket API
- **State**: React Hooks

## Key Features
- Real-time project planting visualization
- WebSocket-based live updates
- Permanent project creation (not temporary)
- GitHub org and deployment links
- Project ownership transfer option
- Responsive design
- Modern UI/UX
